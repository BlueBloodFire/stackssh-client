基于真实代码结构的快速视觉评审原型

StackSSH Client 界面方案

这版原型的目标不是解释功能，而是快速比较产品气质。你可以直接切换主题，观察登录、工作台、文件、Git、Extensions、设置这些核心画面在不同风格下的层级、对比和品牌感。

3 套主题 默认灰色、GitHub 白、GitHub 黑，方便横向评审。
6 个主屏 登录、工作台、文件、Git、Extensions、设置全部同步换肤。
双入口切换 顶部全局切换和设置页内切换都可用，符合产品预览和评审两种场景。

这轮重点看什么

  • 默认灰色是不是足够像桌面开发工具，而不是普通后台。
  • GitHub 白主题是否够干净，但不会失去专业工具感。
  • GitHub 黑主题是否让终端、AI 和文件区的层级更稳定。
  • 右侧 AI 面板是否足够突出，还是还要继续强化。
  • 哪个主题最适合作为后续 React 页面改造的主方向。
